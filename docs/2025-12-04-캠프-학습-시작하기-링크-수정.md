# 캠프 학습 시작하기 링크 수정

## 작업 일시
2025-12-04

## 문제 상황
캠프 목록에서 "학습 시작하기" 버튼 클릭 시 일반 모드의 학습관리 메뉴(`/today`)로 이동하고 있었음.
캠프 모드의 학습관리 페이지(`/camp/today`)로 이동해야 함.

## 수정 내용

### 파일
- `app/(student)/camp/_components/CampInvitationActions.tsx`

### 변경 사항
1. **"학습 시작하기" 버튼** (active 상태)
   - 변경 전: `href="/today"`
   - 변경 후: `href="/camp/today"`

2. **"학습 재개하기" 버튼** (paused 상태)
   - 변경 전: `href="/today"`
   - 변경 후: `href="/camp/today"`
