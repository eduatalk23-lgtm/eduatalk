## 출석 관리 개요

출석 관리 기능은 다음을 포함합니다:

- QR 코드 생성 및 관리
- SMS 알림 설정
- 출석 기록 조회
- SMS 발송 로그 확인

---

## QR 코드 관리

### QR 코드 생성

1. 관리자 페이지 → **출석 관리** → **QR 코드 관리** (`/admin/attendance/qr-code`)
2. "새 QR 코드 생성" 버튼 클릭
3. 만료 시간 설정
4. 생성 완료

### QR 코드 관리

- **활성화/비활성화**: QR 코드 사용 여부 제어
- **만료 시간**: QR 코드 유효 기간 설정
- **사용 통계**: QR 코드 사용 횟수 및 마지막 사용 시간 확인

### 주의사항

- QR 코드는 학원별로 독립적으로 관리됩니다
- 만료된 QR 코드는 자동으로 비활성화됩니다
- QR 코드는 보안을 위해 정기적으로 갱신하는 것을 권장합니다

## SMS 설정 관리

### 설정 위치

관리자 페이지 → **출석 관리** → **출석 설정** (`/admin/attendance/settings`)

### 주요 설정

자세한 내용은 [SMS 설정 가이드](./sms-settings-guide.md)를 참고하세요.

### 설정 저장

설정 변경 후 "저장" 버튼을 클릭하여 변경사항을 적용합니다.

## 출석 기록 조회

### 출석 기록 조회 방법

1. 관리자 페이지 → **출석 관리** → **출석 기록** (`/admin/attendance`)
2. 날짜 범위 선택
3. 학생별 필터링 (선택사항)
4. 조회

> **참고**: 출석 관리 메뉴는 이제 "학생 관리" 카테고리에서 독립된 별도 카테고리로 분리되었습니다.

### 출석 기록 정보

- 출석 날짜
- 입실 시간 및 방법
- 퇴실 시간 및 방법
- 출석 상태 (출석/결석/지각/조퇴/공결)
- 비고

## SMS 로그 확인

### SMS 로그 페이지

관리자 페이지 → **출석 관리** → **SMS 로그** (`/admin/attendance/sms-logs`)

### 확인 가능한 정보

- 발송 시간
- 학생명
- 수신자 번호 (마스킹 처리: `010-****-1234`)
- SMS 타입 (입실/퇴실/결석/지각)
- 발송 상태 (성공/실패/대기)
- 메시지 내용
- 에러 메시지 (실패 시)

### 필터링 및 정렬

- 날짜 범위 필터
- 학생별 필터
- 상태별 필터 (성공/실패/대기)
- SMS 타입별 필터
- 최신순 정렬

### 통계 정보

- 전체 로그 수
- 성공 건수
- 실패 건수
- 대기 건수

## 문제 해결

### Q1. 학생이 QR 코드를 스캔했는데 출석이 안 됩니다

**확인 사항**:
1. QR 코드가 활성화되어 있는지 확인
2. QR 코드 만료 시간 확인
3. 학생이 올바른 학원의 QR 코드를 스캔했는지 확인
4. SMS 로그에서 에러 메시지 확인

### Q2. SMS가 발송되지 않습니다

**확인 사항**:
1. SMS 설정 확인 (학원 기본 설정 및 학생별 설정)
2. "학생 직접 체크인 시 발송" 설정 확인
3. SMS 로그에서 발송 실패 원인 확인
4. 학생 프로필의 전화번호 확인

### Q3. 출석 기록이 중복으로 생성됩니다

**확인 사항**:
1. 출석 기록 검증 로직이 자동으로 중복을 방지합니다
2. 같은 날짜에 중복 입실/퇴실 시도 시 에러 메시지 확인

### Q4. QR 코드를 새로 생성해야 합니다

**방법**:
1. 기존 QR 코드 비활성화 (선택사항)
2. 새 QR 코드 생성
3. 학생들에게 새 QR 코드 공유
