## 기본 사용법

### 출석 체크 페이지 접근

1. 학생 계정으로 로그인
2. 사이드바에서 "출석 체크" 메뉴 클릭
3. 출석 체크 페이지로 이동

### 출석 방법 선택

출석 체크 페이지에서 다음 중 하나를 선택할 수 있습니다:

- **QR 코드**: 학원의 QR 코드를 스캔하여 출석
- **위치 기반**: 현재 위치를 확인하여 출석
- **수동**: 관리자가 직접 출석 처리

## QR 코드 출석

### 입실 방법

1. 출석 체크 페이지에서 "QR 코드" 선택
2. 카메라 권한 허용 (최초 1회)
3. 학원의 QR 코드를 카메라에 비춤
4. 자동으로 QR 코드 인식 및 입실 처리
5. "출석 체크가 완료되었습니다!" 메시지 확인

### 퇴실 방법

- **QR 입실한 경우**: 퇴실 시에도 QR 코드 스캔 필수
  - 출석 상태 화면에 "퇴실 시 QR 코드 스캔이 필요합니다" 안내 표시
  - QR 코드를 스캔하여 퇴실 처리

### 주의사항

- QR 코드는 학원 내에서만 유효합니다
- 다른 학원의 QR 코드는 사용할 수 없습니다
- QR 코드가 만료되었거나 비활성화된 경우 출석이 불가능합니다

## 위치 기반 출석

### 입실 방법

1. 출석 체크 페이지에서 "위치 기반" 선택
2. 위치 정보 접근 권한 허용 (최초 1회)
3. 현재 위치 확인 후 입실 처리
4. 학원 반경 내에 있어야 출석 인정

### 퇴실 방법

- 위치 입실한 경우: 버튼 클릭으로 퇴실 처리 가능
- 위치 확인 또는 수동 퇴실 선택 가능

### 주의사항

- 위치 정보 접근 권한이 필요합니다
- 학원 반경(기본 100m) 내에 있어야 출석이 인정됩니다
- 위치 정보가 정확하지 않을 수 있습니다

---

## 수동 출석

### 입실 방법

- 관리자가 직접 출석 처리
- 학생 앱에서는 수동 출석 불가

### 퇴실 방법

- 수동 입실한 경우: 버튼 클릭으로 퇴실 처리

---

## 퇴실 처리

### 퇴실 방법 결정

퇴실 방법은 입실 방법에 따라 자동으로 결정됩니다:

- **QR 입실** → **QR 퇴실 필수**
- **위치 입실** → **버튼 클릭 퇴실** (또는 위치 확인)
- **수동 입실** → **버튼 클릭 퇴실**

### 퇴실 처리

1. 출석 상태 화면에서 퇴실 버튼 클릭
2. QR 입실인 경우: QR 코드 스캔
3. 퇴실 시간 기록 및 완료

---

## 출석 상태 확인

### 출석 상태 화면

출석 체크 페이지 상단에 오늘의 출석 상태가 표시됩니다:

- **출석 체크 전**: 회색 점
- **출석 완료**: 초록색 점
- **입실 시간**: 입실 시간 및 방법 표시
- **퇴실 시간**: 퇴실 시간 및 방법 표시
- **QR 스캔 필요 안내**: QR 입실 시 퇴실 전 안내 표시
- **SMS 발송 상태**: 입실/퇴실 SMS 발송 상태 표시

### SMS 발송 상태

- **발송 완료**: 초록색 체크 아이콘
- **전달 완료**: 초록색 체크 아이콘
- **발송 실패**: 빨간색 X 아이콘 (에러 메시지 표시)
- **발송 대기**: 노란색 시계 아이콘

## 자주 묻는 질문

### Q1. QR 코드를 스캔했는데 출석이 안 됩니다

- 다른 학원의 QR 코드인지 확인
- QR 코드가 만료되었는지 확인
- 네트워크 연결 상태 확인
- 카메라 권한이 허용되었는지 확인

### Q2. 위치 기반 출석이 안 됩니다

- 위치 정보 접근 권한이 허용되었는지 확인
- 학원 반경 내에 있는지 확인
- GPS 신호가 약한 곳에서는 정확도가 떨어질 수 있음

### Q3. 퇴실 시 QR 코드를 스캔해야 하나요?

- QR 코드로 입실한 경우에만 QR 코드로 퇴실해야 합니다
- 위치 기반 또는 수동 입실한 경우는 버튼 클릭으로 퇴실 가능

### Q4. SMS가 발송되지 않습니다

- 관리자 페이지에서 SMS 설정 확인
- "학생 직접 체크인 시 발송" 설정이 활성화되어 있는지 확인
- SMS 로그 페이지에서 발송 실패 원인 확인
