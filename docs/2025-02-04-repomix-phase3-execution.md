다음 경로의 파일들을 분석했습니다:

- `app/(student)/plan/` - 학습 계획 생성/관리

- `lib/plan/` - 플랜 관련 비즈니스 로직
- `lib/scores/` - 성적 처리 로직
- `lib/metrics/` - 학습 지표 계산
