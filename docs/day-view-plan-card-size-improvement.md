# 일별 캘린더 플랜 카드 크기 개선

## 작업 개요

일별 캘린더(DayView)에서 사용되는 플랜 카드의 내부 요소들을 영역에 맞게 크기를 키워 가독성과 사용성을 개선했습니다.

## 변경 사항

### 1. 패딩 증가
- **이전**: `p-3` (12px)
- **변경**: `p-4 md:p-5` (16px / 20px)
- 반응형으로 모바일과 데스크톱에서 다른 패딩 적용

### 2. 텍스트 크기 증가

#### 상태 뱃지
- **이전**: `text-[10px]`, `px-2 py-0.5`
- **변경**: `text-xs`, `px-3 py-1`

#### 시간 표기
- **이전**: `text-[10px]`, 아이콘 `h-2.5 w-2.5`
- **변경**: `text-xs`, 아이콘 `h-3.5 w-3.5`

#### 교과/과목
- **이전**: `text-[10px]`
- **변경**: `text-xs`

#### 제목
- **이전**: `text-sm`
- **변경**: `text-base md:text-lg`

#### 회차
- **이전**: `text-xs`
- **변경**: `text-sm`

#### 학습 범위
- **이전**: `text-[10px]`
- **변경**: `text-xs md:text-sm`

#### 진행률
- **이전**: `text-sm`
- **변경**: `text-base md:text-lg`

### 3. 아이콘 크기 증가
- **콘텐츠 아이콘**: `text-base` → `text-xl md:text-2xl`
- **연결 아이콘**: `size={14}` → `size={16}`
- **시계 아이콘**: `h-2.5 w-2.5` → `h-3.5 w-3.5`

### 4. 간격 증가
- **요소 간 간격**: `gap-1.5` → `gap-2 md:gap-2.5`
- **컬럼 간 간격**: `gap-3` → `gap-4`
- **진행률 간격**: `gap-1` → `gap-1.5`

### 5. 진행률 바 크기 증가
- **높이**: `h-2` → `h-3`
- **너비**: `w-16` → `w-20 md:w-24`

### 6. 연결 아이콘 위치 조정
- **이전**: `top-2 right-2`
- **변경**: `top-3 right-3 md:top-4 md:right-4`

## 반응형 디자인

모바일과 데스크톱에서 다른 크기를 적용하여 각 화면 크기에 최적화:

- **모바일**: 기본 크기 (더 작은 패딩, 텍스트)
- **데스크톱 (md 이상)**: 더 큰 크기 (더 큰 패딩, 텍스트, 진행률 바)

## 사용자 경험 개선

1. **가독성 향상**: 텍스트와 아이콘이 더 크게 표시되어 정보를 더 쉽게 파악 가능
2. **터치 친화적**: 모바일에서 터치하기 쉬운 크기로 조정
3. **시각적 계층 구조**: 크기 차이로 정보의 중요도가 더 명확하게 구분됨
4. **일관성**: 일별 캘린더의 넓은 영역에 맞는 적절한 크기로 조정
