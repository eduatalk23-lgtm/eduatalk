UI 시안 설명

1. 서브탭 추가

기존 "성적 분석" / "성적 입력" 사이에 "성적 조회" 서브탭 추가

2. 필터 바

내신/모의고사 토글 + 학년/학기/교과군 드롭다운. CSV 다운로드 버튼 포함

3. 요약 카드

필터된 범위의 총 과목수, 평균 등급, 평균 원점수, 총 이수단위, 추정 백분위 평균

4. 성적 테이블

  • 교과군별 색상 뱃지로 시각 구분
  • 성취평가제 과목은 "P/F" 태그 + 원점수/평균/등급 비활성화
  • 추정 백분위, 변환등급은 자동 산출값 (computation.ts)
  • 체크박스로 다중 선택 → 일괄 삭제

5. 상세/편집 패널

  • 행 클릭 시 테이블 아래에 인라인 패널 펼침
  • 좌측: 모든 입력 필드 편집 가능 (성취도 비율은 아코디언)
  • 우측: 산출 엔진 결과 실시간 미리보기 (읽기전용)
  • 산출 수식 과정도 표시 (Z-score, 백분위, 조정등급 계산 과정)
  • 저장/취소/삭제 버튼

6. 모의고사 뷰

토글 전환 시 시험명/시험일자/원점수/표준점수/백분위/등급 컬럼으로 변경
