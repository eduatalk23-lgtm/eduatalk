# 🧪 Wizard Phase 4 테스트 가이드

**작성일**: 2025년 11월 29일  
**Phase**: 4.5 - 테스트 가이드  
**대상**: Step6Simplified 및 Summary 컴포넌트

---

## 📋 테스트 개요

### 목적
Step6Simplified와 8개의 Summary 컴포넌트가 정상적으로 작동하는지 수동으로 테스트

### 범위
- ✅ 공통 컴포넌트 (3개)
- ✅ 섹션별 Summary (5개)
- ✅ Step6Simplified 통합
- ✅ PlanGroupWizard 통합

### 예상 소요 시간
**2시간** (47개 테스트 케이스)

---

## 🚀 테스트 환경 설정

### 1. 개발 서버 시작

```bash
npm run dev
```

### 2. 테스트 계정 준비

- **학생 계정**: 일반 플랜 생성용
- **관리자 계정**: 캠프 플랜 관리용

### 3. 브라우저 설정

- Chrome/Firefox 최신 버전
- 개발자 도구 열기 (F12)
- Console 탭 확인 (에러 감시)
