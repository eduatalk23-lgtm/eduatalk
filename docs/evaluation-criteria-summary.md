## 1. 좋은 세특의 8단계 흐름

AI가 세특을 **작성·평가·방향 제안**할 때 모두 이 흐름을 기준으로 합니다.

| 단계 | 이름 | AI 확인 기준 |
|---|---|---|
| ① | 지적호기심/의문 | 수업에서 배운 개념에서 출발한 **구체적** 의문 |
| ② | 주제 선정(진로 연결) | 교과에서 자연스럽게 확장된, 학생 수준에 적합한 주제 |
| ③ | 탐구 내용/이론 | 교과 이론을 정확하게 적용한 단계적·논리적 탐구 |
| ④ | 참고문헌/독서 | 출처가 탐구와 실제로 연결되는 독서·자료 활용 |
| ⑤ | 결론(해결/제언/고안) | 한계 인식 + 해결방안/정책 제언/창의적 고안 → **+@ 가산 요소** |
| ⑥ | 교사 관찰(구체적 근거) | 상투적 표현이 아닌, 구체적 관찰 근거 기반 서술 |
| ⑦ | 성장 서사 | 성취과정/발전가능성/잠재력이 드러남 |
| ⑧ | 오류→재탐구 순환 | 한계 인식 → 원인 분석 → 재적용 → 깨달음 → **+@ 정성적 가산** |

### ⑤⑧의 본질 (모델 비교 보고서 컨설턴트 확인)

> ⑤결론과 ⑧오류→재탐구는 **반드시 있어야 상위권에 합격하는 것이 아님**. +@의 부분이다.
> 단, 진로 관련 교과·동아리·진로활동에서는 이 부분에 비중을 두고 봐야 한다.
> ⑧이 있으면 입학사정관이 **탐구 과정을 쫓아가며 학생의 성장을 높이 평가**한다.
> **점수 기반이 아닌 정성적 가산** — 입학사정관의 눈에 띄는 효과.

### 8단계 충족도 → 전형 적합성

| 충족도 | 전형 추천 |
|---|---|
| **80%+** | 서류 100% 전형(학종) 도전 가능 — 생기부만으로 역량이 충분히 드러남 |
| **60~70%** | 학종 가능, 전공 관련 교과·창체(동아리/진로) 중심 충족 확인 필요 |
| **~50%** | 중상위권 학종 합격 어려움. 교과전형 주전형, 면접전형·수능최저 전형 우선 |
| **50% 미만** | 내신이 좋으면 교과전형, 아니면 1차 정량→2차 면접 단계별 전형 추천 |

### "~임을 설명함" 패턴 처리 (모델 비교 보고서 피드백)

> 학생부는 **학교 선생님**이 작성하며, 선생님마다 기재 역량과 관심도가 천차만별이다.
> "~임을 설명함"으로 끝나는 탐구를 **무조건 결론 미완으로 판단하면 안 됨**.
> 앞의 기재 내용에서 실제 수행이 추론 가능하면 → **면접에서 검증할 포인트**로 분류.
> 면접 대비 질문으로 확인 요청 또는 준비사항을 학생에게 고지해야 함.

## 2. 대학 수준별 차등 기준

### 진로교과 최소 충족 단계

| 대학 수준 | 최소 단계 | 참고문헌(④) | 진로 연결 비율 |
|---|---|---|---|
| **SKY카+** (상위권) | ①②③④⑤ (5개) | **필수** | 가능한 **모든** 진로 관련 교과 |
| **서울 15대학** (중상위) | ①②③⑤ (4개) | 권장 | 3~4과목 |
| **하위권** | ①②③⑤ (4개) | 선택 | 3~4과목 |

### 진로교과 vs 비진로교과

| 구분 | 기준 |
|---|---|
| **진로교과** | 8단계 흐름 충족 + 교과 단원-진로 호기심 연결. 탐구 깊이 기대치 높음 |
| **비진로교과** | 교과 역량 중심이 정상. 순수 호기심 탐구가 오히려 좋은 평가. 진로 연결 불필요 |
| **금지** | 모든 교과에 동일 진로 키워드 강제 삽입 = 입학사정관 감점 (F16, severity: major) |

### 평가 vs 설계 관점 분리 (모델 비교 보고서 피드백)

> **평가(현재 기록 분석)**: 교사 기재 관용 원칙 적용. 탐구 폭과 방향이 맞으면 인정.
> **설계(다음 학기 방향)**: 8단계 이상향 적용. 각 탐구의 결론·방법론이 구체적이도록 가이드.
> 상위권 대학일수록 깊이(결론·방법론) 중시, 중하위권은 폭(주제 다양성·방향)도 충분.

### 내신-세특 교차 검증 (필수, 모델 비교 보고서 피드백)

> **내신 따로, 세특 따로 평가하면 안 됨.**
> 세특에서 다룬 주제와 탐구역량이 관련 교과 내신 성적에 부합하는지 반드시 비교.
> 예: 내신 4등급인데 세특에 "우수한 학업 성취" → 모순 또는 과장 → needs_review.
> 단, 학업태도는 성실하게 참여한 것을 선생님이 써주신 것으로 인정 가능.

## 3. 내신 등급별 탐구 난이도

| 9등급 | 5등급 | 난이도 | AI 기대 수준 |
|---|---|---|---|
| 1~2등급 | A | 심화+확장 | 교과 심화 이론 + 선행연구 + 실험 설계 + 사회적 확장. **대학 1~2학년 개론** 수준 연결 가능 |
| 3~4등급 | B~C | 발전 | 교과 기반 자연스러운 확장. 단계적 탐구 + 명확한 결론. 전공 개론서 이하 |
| 5등급 이하 | D~E | 기본 | 핵심 개념 이해·적용. 태도·노력 중심 |

- 3~4등급은 하나로 묶어도 적절 (컨설턴트 확인)
- **P4 경고**: 내신 5등급인데 대학원급 심화 탐구 → 대리작성 의심 (severity: critical)

## 5. 창체 활동유형별 평가

### 가중치: 동아리 = 진로 (동등) > 자율 — 비율 1:1:0.7

| 유형 | 가중치 | 중요도 | 평가 핵심 |
|---|---|---|---|
| **동아리** | 0.14 | 상 | 전공 심화 탐구, 문제 설정→해결, 결과물, 협업 |
| **진로** | 0.14 | 상 | 자기주도 조사/실험, 참여→호기심→심화탐구 |
| **자율** | 0.096 | 하 | 아래 참고 |

### 자율활동 평가 기준 (우선순위)

1. **최선**: 학교 교육프로그램 + **교과 이론 대입** + 사회동향·대응방안 고찰 → **인문학적 성찰**
2. **차선**: 공동체역량(전교회장/부회장/반장 등) + **구체적 기여 사례**
3. **활용**: 다른 영역 기재 수 부족 시 담임 재량으로 진로 탐색/독서 탐구 추가 기재

## 6. 행동특성 평가

### 8개 항목 + 5단계 (2차 피드백)

| 항목 | 설명 | 평가 등급 |
|---|---|---|
| 자기주도성 | 스스로 목표를 세우고 계획·실천 | 매우 우수 ~ 매우 미흡 (5단계) |
| 갈등관리 | 갈등 상황에서 합리적으로 해결 | 〃 |
| 리더십 | 공동체를 이끌고 방향 제시 | 〃 |
| **타인존중·배려** | 다양성 인정, 타인 배려, 봉사와 공감 실천 | 〃 |
| 성실성 | 책임감, 꾸준한 노력 | 〃 |
| 규칙준수 | 공동체 규범, 질서 유지 | 〃 |
| **회복탄력성** (신규) | 실패나 어려움 후 다시 도전하고 성장 | 〃 |
| **지적호기심** (신규) | 스스로 질문을 만들고 답을 찾아가는 탐구적 태도 | 〃 |

### 행특의 본질 (컨설턴트 관점)

> 행동특성은 **1년간 학생을 지켜본 교사의 관찰**로서 **발전가능성과 잠재력**을 보여주는 추천서 성격.
> 평가 키워드가 **구체적 사례**로 쓰여 자연스럽게 **장면이 상상**되면 가장 좋은 행특.

## 7. 품질 5축 평가

### 가중치 (합계 100점)

| 축 | 한국어명 | 가중치 | 평가 내용 |
|---|---|---|---|
| specificity | 구체성 | **25** | 구체적 사례·근거·성과 포함 정도 |
| coherence | 논리적 연결 | **15** | 8단계 흐름의 논리적 연결 + 학년별 성장 |
| depth | 탐구 깊이 | **25** | 심층 분석, 교과 연계, 확장적 사고 |
| grammar | 문법/표현 | **10** | 문법·맞춤법·표현 적절성 |
| scientificValidity | **연구 정합성** | **25** | 아래 계열별 분기 |

### 연구 정합성 — 계열별 평가 기준

**이공계 (과학·수리적 정합성):**
- 개념 정확성, 실험·모델링 설계 타당성, 결론 비자명성, 원리 해석 오류

**인문·사회계 (사회연구 정합성):**
- 문제 정의 → 연구 질문 → 연구방법(양적: 질문지법·통계 / 질적: 면접법·참여관찰법·문헌연구법) → 자료분석 → 결론과 제언
- 한계점 인식 및 대응방안 논리적 도출

**고등학생 수준 기준** (2차 피드백):
- 양적연구: 표본 규모보다 **학생이 직접 설계·수행·결과 도출**한 것이 핵심
- 문헌연구: 책 + **국내외 연구논문, 학술지** 참고 정도면 충분

**우수 사례 (2차 피드백 — 프롬프트 few-shot용):**
1. 뉴스 감성 점수 + 주가 상관관계 분석 → 한계 도출 (금융/IT 융합)
2. 넛지 독서 → 식사 선택 실험 → 200명 확장 연구 (행동경제학)
3. 문학작품 + 경제지표(지니계수, 로렌츠 곡선) 융합 → 소득불평등 시각화 (인문+경제)

## 9. 비진로교과 세특 평가 기준 (2차 피드백 신설)

### 국어 교과 (과목별)

| 과목 | 평가 요소 |
|---|---|
| **문학** | 독자적 해석, 작품 간 비교(심경변화·상황 종합), 현대사회 기준 재해석, 창의적 결말 고안, 함축적 의미 해석, 비판적 사고 |
| **독서와 작문** | 자료해석능력, 독해능력, 비판적 사고, 논리추론능력 |
| **화법과 언어** | 품사·문장 구조 활용, 어휘·문장 선택, 담화 구성, 공적 의사소통(토의·토론·연설·협상) |

### 영어 교과

- 활동중심·모둠협력 수업 내 대화·발표·작업이 평가 근거
- 독해: 세부정보, 요지, 논리적 관계, 추론, 어휘·구문 이해
- 읽기·쓰기·듣기·말하기 종합 평가

### 비진로교과에서 ⑧ 오류→재탐구

> 필수가 아님. 단, 있으면 **학업역량에서 좋은 평가** + **진로 교과 종합평가에서 탐구력 이해 근거**로 작용.
