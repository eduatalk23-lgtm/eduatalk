# 스케줄 미리보기 페이지 검증 로직 제거

## 작업 일자
2025-01-27

## 작업 내용
스케줄 미리보기 페이지(Step 3)에서 다음 버튼 클릭 시 실행되던 검증 로직을 제거했습니다.

## 변경 사항

### 수정된 파일
- `app/(student)/plan/new-group/_components/PlanGroupWizard.tsx`

### 변경 내용
`handleNext` 함수에서 Step 3일 때 검증 로직을 건너뛰도록 수정했습니다.

**변경 전:**
```typescript
if (!validateStep(currentStep)) {
  return;
}
```

**변경 후:**
```typescript
// Step 3 (스케줄 미리보기)에서는 검증 로직 건너뛰기
if (currentStep !== 3) {
  if (!validateStep(currentStep)) {
    return;
  }
}
```

## 이유
스케줄 미리보기 페이지는 단순히 설정된 블록과 제외일을 기반으로 생성된 스케줄을 확인하는 단계이므로, 다음 단계로 진행하기 위한 검증이 필요하지 않습니다.

## 영향 범위
- Step 3 (스케줄 미리보기)에서 다음 버튼 클릭 시 검증 없이 바로 다음 단계로 이동 가능
- 다른 단계의 검증 로직은 그대로 유지
