## 구현 내용

### 1. PlanDetailInfo 컴포넌트
- **위치**: `app/(student)/today/_components/PlanDetailInfo.tsx`
- **기능**: 헤더 아래에 학습 범위, 회차, 블록 수를 아이콘 기반으로 표시
- **표시 정보**:
  - 📖 학습 범위: `p.50 ~ p.75` (책) 또는 `00:10 ~ 00:30` (강의)
  - 🔄 회차: `1회차`
  - 📚 블록: `2개` (블록이 2개 이상일 때만 표시)

### 2. TimeCheckSection 컴포넌트
- **위치**: `app/(student)/today/_components/TimeCheckSection.tsx`

- **표시 정보**:

  - 일시정지 정보 (횟수 및 시간)
  - 실시간 타이머 (진행 중인 경우)
  - 완료 상태 표시

## 주요 기능

### 실시간 타이머
- 진행 중인 플랜의 경우 1초마다 경과 시간을 업데이트
- 일시정지 중일 때는 타이머가 멈춤
- 순수 학습 시간 계산 (일시정지 시간 제외)

### 시간 통계
- 총 학습 시간: 모든 블록의 학습 시간 합산
- 순수 학습 시간: 일시정지 시간을 제외한 실제 학습 시간
- 일시정지 정보: 일시정지 횟수 및 총 일시정지 시간

### 학습 범위 표시
- 첫 번째 블록의 시작 ~ 마지막 블록의 종료 범위 표시
- 책인 경우: `p.50 ~ p.75`
- 강의인 경우: `00:10 ~ 00:30`
