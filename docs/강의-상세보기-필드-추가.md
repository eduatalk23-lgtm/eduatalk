# 강의 상세보기 필드 추가 작업

## 작업 일시
2025-01-XX

## 문제 상황
강의 상세보기 페이지에서 최근에 추가된 필드들(출처 URL, 강사, 강의 대상, 강의 유형 등)이 표시되지 않았습니다.

## 수정 내용

### 1. 학생 강의 상세보기 페이지 수정
**파일**: `app/(student)/contents/lectures/[id]/page.tsx`

- select 쿼리에 누락된 필드 추가:
  - `instructor` (강사)
  - `source_url` (출처 URL)
  - `grade_min` (최소 학년)
  - `grade_max` (최대 학년)
  - `content_category` (강의 유형)

- 마스터 강의 정보 참조 로직 추가:
  - 학생 강의에 필드가 없을 경우 마스터 강의에서 정보를 가져와서 표시하도록 수정

### 2. LectureInfoSection 컴포넌트 수정
**파일**: `app/(student)/contents/lectures/[id]/_components/LectureInfoSection.tsx`

- 누락된 필드 표시 추가:
  - 강사
  - 강의 대상 학년 (formatGradeLevel 유틸리티 사용)
  - 강의 유형
  - 출처 URL (링크로 표시)

- 필요한 유틸리티 함수 import 추가:
  - `formatGradeLevel`
  - `secondsToMinutes`

### 3. 타입 정의 수정
**파일**: `app/types/content.ts`

- `Lecture` 타입에 필드 추가:
  ```typescript
  instructor?: string | null;
  source_url?: string | null;
  grade_min?: number | null;
  grade_max?: number | null;
  content_category?: string | null;
  ```

### 4. LectureDetailTabs 컴포넌트 타입 수정
**파일**: `app/(student)/contents/lectures/[id]/_components/LectureDetailTabs.tsx`

- `LectureDetailTabsProps` 타입에 필드 추가

### 5. 마스터 강의 복사 함수 수정
**파일**: `lib/data/contentMasters.ts`

- `copyMasterLectureToStudent` 함수에서 필드 복사 추가:
  - `instructor`
  - `source_url`
  - `grade_min`
  - `grade_max`
  - `content_category`

## 참고 사항

- 마스터 강의 상세보기 페이지(`master-lectures/[id]/page.tsx`)는 이미 모든 필드가 표시되고 있었습니다.
- 학생 강의에 필드가 없을 경우 마스터 강의 정보를 참조하여 표시하도록 구현했습니다.
- `formatGradeLevel` 유틸리티 함수를 사용하여 학년 범위를 포맷팅합니다 (예: "1-2학년").

## 테스트 항목

- [ ] 학생 강의 상세보기에서 출처 URL이 표시되는지 확인
- [ ] 학생 강의 상세보기에서 강사가 표시되는지 확인
- [ ] 학생 강의 상세보기에서 강의 대상 학년이 표시되는지 확인
- [ ] 학생 강의 상세보기에서 강의 유형이 표시되는지 확인
- [ ] 마스터 강의에서 복사한 강의의 경우 필드가 제대로 복사되는지 확인
- [ ] 마스터 강의 정보 참조가 제대로 동작하는지 확인
