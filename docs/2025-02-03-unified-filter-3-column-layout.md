# UnifiedContentFilter 3열 레이아웃 재구성

## 📋 작업 개요

`UnifiedContentFilter` 컴포넌트의 레이아웃을 3열 그리드로 재구성하여 필터 항목을 더 체계적으로 배치했습니다.

## 🎨 레이아웃 구조

### 변경 전
- `flex flex-wrap` 레이아웃으로 자동 줄바꿈
- 필터 항목들이 가로로 나열되다가 공간이 부족하면 자동으로 다음 줄로 이동

### 변경 후
- `grid grid-cols-1 md:grid-cols-3` 레이아웃으로 3열 구조
- 모바일: 1열 (세로 배치)
- 데스크톱: 3열 (가로 배치)

## 📐 열 구성

### 1열: 계층형 필터
- **개정교육과정**: 교육과정 선택
- **교과**: 교과 선택 (개정교육과정 선택 후 활성화)
- **과목**: 과목 선택 (교과 선택 후 활성화)

### 2열: 콘텐츠 속성 필터
- **출판사** (교재용): 교재의 출판사 선택
- **플랫폼** (강의용): 강의의 플랫폼 선택
- **난이도**: 콘텐츠의 난이도 선택

### 3열: 검색 및 액션
- **제목 검색**: 콘텐츠 제목으로 검색
- **정렬**: 정렬 옵션 선택
- **검색 버튼**: 필터 적용
- **초기화 버튼**: 필터 초기화

## 🔧 기술적 변경 사항

### 레이아웃 변경
```typescript
// 변경 전
className={`flex flex-wrap items-end gap-4 ${className}`}

// 변경 후
className={`grid grid-cols-1 md:grid-cols-3 gap-4 items-end ${className}`}
```

### 구조 변경
- 각 열을 `div`로 그룹화하여 세로 배치 (`flex flex-col gap-4`)
- 1열, 2열, 3열로 명확하게 구분
- 검색/초기화 버튼을 같은 행에 배치 (`flex gap-2`)

### 반응형 디자인
- 모바일 (`grid-cols-1`): 모든 필터가 세로로 배치
- 데스크톱 (`md:grid-cols-3`): 3열 그리드로 배치

## 📝 수정된 파일

- `components/filters/UnifiedContentFilter.tsx`
  - 레이아웃을 `flex flex-wrap`에서 `grid grid-cols-1 md:grid-cols-3`로 변경
  - 필터 항목을 3열로 그룹화
  - 검색/초기화 버튼을 같은 행에 배치

## 🎯 결과

- 필터 항목이 논리적으로 그룹화되어 가독성 향상
- 계층형 필터(1열), 속성 필터(2열), 검색/액션(3열)로 명확하게 구분
- 반응형 디자인으로 모바일과 데스크톱 모두에서 최적화된 레이아웃 제공
