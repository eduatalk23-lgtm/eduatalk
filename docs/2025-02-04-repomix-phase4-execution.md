## 🔍 주요 분석 내용

### 1. 콘텐츠 관리 (`app/(student)/contents/`)

- 마스터 북/강의 관리
- 콘텐츠 필터링 및 검색
- 콘텐츠 상세 정보

### 2. 오늘의 학습 (`app/(student)/today/`)

- 오늘의 학습 계획 표시
- 학습 진행 상황 추적
- 일일 학습 목표

### 3. 대시보드 (`app/(student)/dashboard/`)

- 학습 통계 표시
- 진행 중인 플랜 요약
- 성적 트렌드
- 학습 시간 분석

### 4. 학습 분석 (`app/(student)/analysis/`)

- 학습 패턴 분석
- 성취도 분석
- 시간대별 학습 분석
- 과목별 분석

### 5. 블록 관리 (`app/(student)/blocks/`)

- 학습 블록 생성 및 관리
- 블록 스케줄링
- 블록 완료 처리

### 6. 캠프 기능 (`app/(student)/camp/`)

- 캠프 참여 관리
- 캠프 일정 관리
- 캠프 진행 상황

### 7. 데이터 페칭 (`lib/data/`)

- 플랜 그룹 데이터
- 콘텐츠 마스터 데이터
- 성적 데이터
- 캠프 데이터

### 8. 추천 엔진 (`lib/recommendations/`)

- 콘텐츠 추천 알고리즘
- 학습 패턴 기반 추천
