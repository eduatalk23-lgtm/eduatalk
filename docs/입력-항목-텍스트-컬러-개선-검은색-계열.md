# 입력 항목 및 텍스트 컬러 개선 (검은색 계열)

**작업 일자**: 2025년 1월  
**작업 범위**: 입력 필드 및 관련 텍스트의 가독성 개선 (검은색 계열 적용)

## 문제점

- `globals.css`의 `--text-placeholder`가 `#9ca3af` (gray-400)로 설정되어 있어 placeholder 텍스트가 옅음
- `FormField`의 hint 텍스트가 `text-gray-500`로 설정되어 가독성이 떨어짐
- `TimeRangeInput` 등 일부 컴포넌트에서 직접 회색 텍스트를 사용하여 일관성 부족
- 회색 계열 대신 검은색 계열로 조절하여 더 명확한 가독성 확보 필요

## 개선 방안

### 1. CSS 변수 개선 (`app/globals.css`)

**라이트 모드**:
- `--text-placeholder`: `#9ca3af` (gray-400) → `#4b5563` (gray-600)
- `--text-secondary`: `#6b7280` (gray-500) → `#374151` (gray-700)
- `--text-tertiary`: `#9ca3af` (gray-400) → `#4b5563` (gray-600)

**다크 모드**:
- `--text-secondary`: `#9ca3af` (gray-400) → `#d1d5db` (gray-300)
- `--text-tertiary`: `#6b7280` (gray-500) → `#9ca3af` (gray-400)
- `--text-placeholder`: `#6b7280` (gray-500) → `#9ca3af` (gray-400)
- `--text-disabled`: `#4b5563` (gray-600) → `#6b7280` (gray-500)

### 2. FormField 컴포넌트 개선 (`components/molecules/FormField.tsx`)

- hint 텍스트: `text-gray-500` → `text-gray-800`

### 3. TimeRangeInput 컴포넌트 개선 (`components/ui/TimeRangeInput.tsx`)

- 설명 텍스트: `text-gray-500` → `text-gray-800`
- disabled 텍스트: `text-gray-500` → `text-gray-800`
- 기본값 표시: `text-gray-500` → `text-gray-800`
- 기본값으로 되돌리기 버튼: `text-gray-500 hover:text-gray-700` → `text-gray-800 hover:text-gray-900`

### 4. PlanGroupListItem 개선 (`app/(student)/plan/_components/PlanGroupListItem.tsx`)

- 라벨 텍스트 (목적, 스케줄러, 기간): `text-gray-500` → `text-gray-800`
- 메타 정보 (생성일, 플랜 개수): `text-gray-500` → `text-gray-800`
- 아이콘 색상:
  - 체크박스 아이콘: `text-gray-400` → `text-gray-700`
  - 활성화/비활성화 아이콘: `text-gray-400` → `text-gray-700`
  - 삭제 아이콘: `text-gray-400` → `text-gray-700`

## 적용 원칙

- **Placeholder**: `gray-600` 또는 `gray-700` (검은색에 가까운 진한 회색)
- **Hint/설명 텍스트**: `gray-800` (검은색 계열)
- **라벨/메타 정보**: `gray-800` (검은색 계열)
- **주요 텍스트**: `gray-900` 또는 `text-black` 유지
- **보조 아이콘**: `gray-700` (검은색 계열)

## 영향 범위

- Input/Select 컴포넌트를 사용하는 모든 페이지 (CSS 변수 변경으로 자동 적용)
- FormField를 사용하는 모든 폼
- TimeRangeInput을 사용하는 페이지
- PlanGroupListItem을 사용하는 플랜 목록 페이지

## 결과

- ✅ Placeholder 텍스트가 더 진하게 표시되어 가독성 향상
- ✅ Hint 및 설명 텍스트가 검은색 계열로 변경되어 명확하게 보임
- ✅ 라벨 및 메타 정보가 검은색 계열로 통일되어 일관성 확보
- ✅ 아이콘 색상이 검은색 계열로 조정되어 가독성 개선
- ✅ 전체적으로 텍스트 인식이 쉬워짐
