## 작업 내용

### Phase 6 분석 범위
다음 디렉토리 및 파일들을 분석했습니다:

1. **부모 모듈**
   - `app/(parent)` - 부모 전용 페이지 및 기능

2. **슈퍼 관리자 모듈**
   - `app/(superadmin)` - 슈퍼 관리자 전용 페이지 및 기능

3. **인증 모듈**
   - `app/login` - 로그인 페이지
   - `app/signup` - 회원가입 페이지

4. **Server Actions**
   - `app/actions` - 서버 액션 모듈

5. **API 라우트**
   - `app/api` - RESTful API 엔드포인트

6. **공통 컴포넌트**
   - `components/navigation` - 네비게이션 컴포넌트
   - `components/layout` - 레이아웃 컴포넌트

7. **비즈니스 로직 라이브러리**
   - `lib/domains` - 도메인 로직
   - `lib/coaching` - 코칭 로직
   - `lib/risk` - 리스크 분석
   - `lib/reschedule` - 일정 재조정 로직
