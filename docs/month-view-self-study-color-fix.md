# 월별 플랜 캘린더 자율학습 색상 수정

## 변경 사항

월별 플랜 캘린더의 자율학습 디자인을 주별 플랜 캘린더와 동일하게 맞추기 위해 색상을 변경했습니다.

### 개선 내용

**자율학습 색상 변경**
- 기존: `bg-orange-100 text-orange-800` (주황색)
- 변경: `bg-green-100 text-green-800` (초록색)
- 주별 뷰와 동일한 색상으로 통일

### 변경 파일

- `app/(student)/plan/calendar/_components/MonthView.tsx`
  - 자율학습 타입일 때 초록색 배경/텍스트 적용
  - 점심시간, 이동시간 등 다른 타입은 기존 주황색 유지

### 일관성

이제 월별, 주별, 일별 뷰 모두에서 자율학습이 동일한 초록색(`bg-green-50 border-green-200 text-green-800` 또는 `bg-green-100 text-green-800`)으로 표시됩니다.
