# 일별 플랜 캘린더 아이콘 중복 문제 해결

## 문제점

일별 플랜 캘린더에서 아이콘이 중복되어 표시되는 문제가 있었습니다:

1. **학습 플랜 타임라인 앞에 있는 아이콘과 콘텐츠 아이콘이 중복**
   - 타임라인 헤더에서 `getTimeSlotIcon("학습시간")`이 "📚" 아이콘을 반환
   - `PlanCard` 컴포넌트에서 콘텐츠 타입별 아이콘(📚, 🎧, 📝)을 표시
   - 학습시간 타입의 아이콘("📚")과 book 콘텐츠 아이콘("📚")이 동일하여 중복 표시

2. **시간 학습 블록 앞에 있는 아이콘과 콘텐츠 아이콘이 중복**
   - 동일한 문제로 타임슬롯 아이콘과 콘텐츠 아이콘이 중복 표시

## 해결 방법

학습시간 타입의 아이콘을 "📚"에서 "⏰"로 변경하여 콘텐츠 아이콘과 구분되도록 수정했습니다.

### 변경 파일

- `app/(student)/plan/calendar/_utils/timelineUtils.ts`
  - `getTimeSlotIcon()` 함수에서 "학습시간" 케이스의 반환값을 "📚"에서 "⏰"로 변경
  - 주석 추가: 콘텐츠 아이콘과 구분하기 위해 시계 아이콘 사용

### 영향 범위

다음 컴포넌트들에서 자동으로 반영됩니다:
- `TimelineItem.tsx` - 타임라인 아이템 헤더
- `DayView.tsx` - 일별 뷰의 타임슬롯 표시
- `WeekView.tsx` - 주별 뷰의 타임슬롯 표시
- `DayTimelineModal.tsx` - 타임라인 모달

## 결과

- 타임라인 헤더: ⏰ (시간대를 나타냄)
- 콘텐츠 아이콘: 📚 (책), 🎧 (강의), 📝 (커스텀)
