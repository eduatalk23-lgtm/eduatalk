- 시스템 관리

- 비밀번호 재설정

- 인증 액션

- 데이터 처리 액션

### 5. API 라우트 (`app/api/`)

- RESTful API 엔드포인트
- 외부 시스템 연동
- 웹훅 처리

### 6. 네비게이션 (`components/navigation/`)

- 역할 기반 레이아웃

### 8. 도메인 로직 (`lib/domains/`)

- 도메인별 비즈니스 로직

### 9. 코칭 엔진 (`lib/coaching/`)

- 학습 코칭 알고리즘

### 10. 리스크 분석 (`lib/risk/`)
