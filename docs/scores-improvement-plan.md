# 성적 대시보드 및 입력 페이지 개선 계획

## 📊 현재 상태 분석

### 연결된 페이지 구조
```
/scores/dashboard          - 성적 대시보드 (메인)
/scores                    - 성적 관리 홈
/scores/new                - 구식 성적 입력 (사용 안 함)
/scores/school/[grade]/[semester]/[subject-group]     - 내신 성적 목록
/scores/school/.../new     - 내신 성적 입력
/scores/mock/[grade]/[subject-group]/[exam-type]       - 모의고사 성적 목록
/scores/mock/.../new       - 모의고사 성적 입력
```

### 주요 문제점

#### 1. 사용성 (UX) 문제
- ❌ 성적 입력 폼이 너무 단순하고 직관적이지 않음
- ❌ 필수/선택 필드 구분이 명확하지 않음
- ❌ 입력 검증 피드백이 부족함
- ❌ 대시보드에서 성적 입력으로 이동하는 경로가 복잡함
- ❌ 성적 목록 테이블이 정보 밀도가 낮고 가독성이 떨어짐
- ❌ 대시보드 섹션 간 네비게이션이 부족함

#### 2. 시안성 (UI) 문제
- ❌ 디자인 시스템 컬러/타이포그래피 미적용
- ❌ Spacing-First 정책 미준수 (margin 사용)
- ❌ 카드 컴포넌트 미활용
- ❌ 일관성 없는 버튼 스타일
- ❌ 차트 및 시각화가 기본적인 형태
- ❌ 반응형 디자인이 일부만 적용됨

#### 3. 기능적 문제
- ❌ 대량 입력 기능 없음
- ❌ 성적 비교 기능이 제한적
- ❌ 필터링/정렬 기능 부족
- ❌ 성적 통계 요약이 부족함

## 🎯 개선 계획

### Phase 1: 시안성 개선 (우선순위: 높음)

#### 1.1 디자인 시스템 적용

**대시보드 페이지**
- [ ] 디자인 시스템 컬러 팔레트 정의 및 적용
- [ ] 타이포그래피 시스템 적용 (`text-h1`, `text-h2`, `text-body-1` 등)
- [ ] Card 컴포넌트 활용하여 섹션 재구성
- [ ] Spacing-First 정책 적용 (gap 우선, margin 제거)

**성적 입력 폼**
- [ ] 폼 레이아웃을 Card로 감싸기
- [ ] 입력 필드 그룹화 및 시각적 계층 구조 개선
- [ ] 버튼 스타일 통일 (Primary/Secondary 구분)

#### 1.2 대시보드 시각화 개선

**SummarySection**
- [ ] 통계 카드를 더 시각적으로 개선 (아이콘, 색상 코딩)
- [ ] 추세 표시를 더 명확하게 (화살표 아이콘 대신 차트)
- [ ] 반응형 그리드 개선

**차트 섹션들**
- [ ] 차트 라이브러리 도입 (recharts 또는 chart.js)
- [ ] 인터랙티브 차트로 개선 (호버, 클릭 이벤트)
- [ ] 색상 코딩으로 데이터 구분 명확화

#### 1.3 성적 목록 테이블 개선

- [ ] 테이블을 Card로 감싸기
- [ ] 행 호버 효과 개선
- [ ] 등급에 따른 색상 코딩 (1등급: 파란색, 9등급: 빨간색)
- [ ] 모바일 반응형 테이블 (카드 형태로 변환)

### Phase 2: 사용성 개선 (우선순위: 높음)

#### 2.1 성적 입력 폼 개선

**폼 구조 개선**
- [ ] 단계별 입력 (Step 1: 기본 정보, Step 2: 성적 정보)
- [ ] 필수 필드 명확히 표시 (별표 + 설명)
- [ ] 실시간 검증 피드백
- [ ] 입력 힌트 및 예시 강화

**UX 개선**
- [ ] 자동완성 기능 (과목명, 시험일)
- [ ] 이전 입력값 기억 기능
- [ ] 빠른 입력 모드 (최소 필드만 입력)
- [ ] 대량 입력 기능 (CSV 업로드 또는 여러 개 한 번에)

**네비게이션 개선**
- [ ] 대시보드에서 직접 성적 입력 버튼 추가
- [ ] 입력 완료 후 선택적 리다이렉트 (목록/대시보드)

#### 2.2 대시보드 네비게이션 개선

- [ ] 섹션별 앵커 링크 추가
- [ ] 스크롤 스파이 (현재 보이는 섹션 하이라이트)
- [ ] 빠른 필터링 (학년, 학기, 과목별)
- [ ] 기간 선택 기능 (최근 1개월, 3개월, 전체)

#### 2.3 성적 목록 개선

- [ ] 필터링 기능 (과목, 학기, 등급 범위)
- [ ] 정렬 기능 (날짜, 등급, 원점수)
- [ ] 검색 기능 (과목명 검색)
- [ ] 일괄 선택 및 삭제

```css
--grade-3: #93c5fd;

--grade-7: #ef4444;  /* 빨간색 */

--grade-9: #991b1b;

/* 상태 색상 */
--trend-up: #10b981;    /* 개선 */
--trend-down: #ef4444;  /* 하락 */
--trend-stable: #6b7280; /* 유지 */
```

### 타이포그래피 적용

```tsx
// 대시보드 제목
<h1 className="text-h1 text-gray-100">성적 대시보드</h1>

// 섹션 제목
<h2 className="text-h2 text-gray-100 mb-4">성적 요약</h2>

// 본문
<p className="text-body-2 text-gray-70">내용</p>
```

## 📝 구체적 개선 작업 목록

### 즉시 적용 가능한 개선 (Quick Wins)

1. **Spacing-First 정책 적용**
   - 모든 `mt-`, `mb-`, `mx-`, `my-` 제거
   - `gap-`, `space-y-`, `p-` 로 대체

2. **Card 컴포넌트 활용**
   - 대시보드 섹션들을 Card로 감싸기
   - 성적 입력 폼을 Card로 감싸기

3. **버튼 스타일 통일**
   - Primary: `bg-indigo-600`
   - Secondary: `border border-gray-300 bg-white`

4. **등급 색상 코딩**
   - 등급별 색상 적용
   - 배지 형태로 표시

5. **반응형 개선**
   - 모바일에서 테이블을 카드 형태로 변환
   - 그리드 레이아웃 반응형 조정

### 중기 개선 작업

1. **차트 라이브러리 도입**
   - recharts 또는 chart.js 선택
   - 기존 차트 컴포넌트 교체

2. **폼 개선**
   - React Hook Form 도입
   - 단계별 입력 구현
   - 실시간 검증

3. **필터링/정렬 기능**
   - 필터 UI 컴포넌트 추가
   - URL 쿼리 파라미터 연동

### 장기 개선 작업

1. **대량 입력 기능**
   - CSV 업로드
   - 여러 성적 한 번에 입력

2. **고급 분석 기능**
   - 목표 설정 및 추적
   - 성적 예측
   - AI 기반 인사이트
