# 템플릿 상세보기 탭 아이콘 제거

## 작업 날짜
2025-12-05

## 작업 개요
플랜 그룹 상세보기 화면의 탭 네비게이션에서 완료 상태를 나타내는 아이콘(체크박스)을 제거하고 텍스트 레이블만 표시하도록 변경했습니다.

## 변경 사항

### 수정된 파일

#### `app/(student)/plan/group/[id]/_components/PlanGroupDetailTabs.tsx`

**변경 내용:**

1. **아이콘 import 제거**
   - `CheckCircle2`, `Circle` 아이콘 import 제거
   - 사용하지 않는 `useState` import 제거

2. **아이콘 렌더링 코드 제거**
   - `tab.completed`에 따른 조건부 아이콘 렌더링 제거 (74-88번 줄)
   - 완료 상태를 나타내는 체크박스 아이콘 제거

3. **스타일 조정**
   - 아이콘과 텍스트 사이 간격용 `gap-2` 클래스 제거
   - 텍스트만 표시하도록 구조 단순화

**변경 전:**
```tsx
import { CheckCircle2, Circle } from "lucide-react";

// ... 중략 ...

{tab.completed ? (
  <CheckCircle2
    className={`h-4 w-4 flex-shrink-0 ${
      currentTab === tab.id ? "text-blue-600" : "text-green-500"
    }`}
    aria-hidden="true"
  />
) : (
  <Circle
    className={`h-4 w-4 flex-shrink-0 ${
      currentTab === tab.id ? "text-blue-600" : "text-gray-700"
    }`}
    aria-hidden="true"
  />
)}
<span className="whitespace-nowrap">{tab.label}</span>
```

**변경 후:**
```tsx
// 아이콘 import 제거

// ... 중략 ...

<span className="whitespace-nowrap">{tab.label}</span>
```
