# 모달 폰트 가독성 개선

**작업일**: 2025-01-30  
**작업 내용**: 모달/다이얼로그에서 사용되는 회색 텍스트 색상을 더 진하게 변경하여 가독성 향상

## 📋 작업 개요

모달 및 다이얼로그 컴포넌트에서 사용되는 회색 텍스트가 너무 옅어서 가독성이 떨어지는 문제를 해결하기 위해, 모든 회색 텍스트 색상을 더 진하게 변경했습니다.

## 🎨 색상 변경 규칙

- `text-gray-500` → `text-gray-700` (설명 텍스트, 보조 정보)
- `text-gray-400` → `text-gray-600` (아이콘, 닫기 버튼)
- `text-gray-600` → `text-gray-700` 또는 `text-gray-800` (본문 텍스트)

## 🎯 개선 효과

### 가독성 향상

1. **텍스트 대비 증가**: 회색 텍스트가 더 진해져서 배경과의 대비가 증가하여 읽기 쉬워졌습니다.

2. **일관성 개선**: 모든 모달/다이얼로그에서 동일한 색상 규칙을 적용하여 일관된 사용자 경험을 제공합니다.

3. **접근성 향상**: 시력이 약한 사용자도 텍스트를 더 쉽게 읽을 수 있습니다.

## 📊 색상 가이드라인

### 텍스트 색상 우선순위

1. **주요 텍스트** (제목, 중요 정보): `text-gray-900`
2. **일반 텍스트** (본문, 설명): `text-gray-700` 또는 `text-gray-800`
3. **보조 텍스트** (메타 정보, 날짜): `text-gray-600`
4. **비활성 텍스트** (disabled): `text-gray-400` (최소한의 사용)

### 아이콘/버튼 색상

- **기본 아이콘**: `text-gray-600`
- **호버 상태**: `text-gray-800`
- **비활성 상태**: `text-gray-400`

## 📝 참고 사항

1. **회색 사용 지양**: 가능한 한 회색 사용을 최소화하고, 중요한 정보는 더 진한 색상(`text-gray-800`, `text-gray-900`)을 사용합니다.

2. **접근성**: WCAG 2.1 AA 기준을 준수하기 위해 텍스트와 배경의 대비 비율이 최소 4.5:1 이상이 되도록 합니다.

3. **일관성**: 프로젝트 전체에 동일한 색상 가이드라인을 적용하여 일관된 디자인을 유지합니다.
