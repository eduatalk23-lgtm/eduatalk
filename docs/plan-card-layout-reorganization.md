# 플랜 카드 레이아웃 재구성

## 변경 사항

PlanCard 컴포넌트의 레이아웃을 재구성하여 정보를 더 명확하게 표시하도록 개선했습니다.

### 새로운 레이아웃 구조

1. **1행: 상태 뱃지 + 시간 표기**
   - 상태 뱃지 (완료/학습 중/대기)
   - 시간 표기 (Clock 아이콘 포함)

2. **2행: 교과 과목**
   - 교과 (contentSubjectCategory)
   - 과목 (contentSubject)

3. **3행: 교재명(또는 강의명) 출판사(플랫폼) 회차**
   - 콘텐츠 아이콘
   - 교재명/강의명 (contentTitle)
   - 회차 (contentEpisode)

4. **4행: 학습 범위**
   - 책: 페이지 범위
   - 강의: 강의 회차
   - 챕터 정보 (있는 경우)

### 제거된 항목

- 중복되는 시간 표기 영역 제거
  - 기존에 1행과 별도로 표시되던 시간 정보를 1행으로 통합
  - `plan.start_time`과 `plan.end_time`이 있을 때만 표시

### 변경 파일

- `app/(student)/plan/calendar/_components/PlanCard.tsx`
  - 레이아웃을 4행 구조로 재구성
  - 상태 뱃지를 1행으로 이동
  - 시간 표기를 1행에 상태 뱃지 옆에 배치
  - 중복 시간 표기 영역 제거
