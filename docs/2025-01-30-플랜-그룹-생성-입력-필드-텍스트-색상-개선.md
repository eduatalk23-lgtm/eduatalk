# 플랜 그룹 생성 입력 필드 텍스트 색상 개선

**작업일**: 2025-01-30  
**작업 내용**: 학생 페이지에서 플랜 그룹 생성 시 모든 입력 필드의 텍스트를 검은색으로 수정하여 가독성 향상

## 📋 작업 개요

플랜 그룹 생성 위저드의 모든 입력 필드(input, textarea, select)에서 입력 텍스트를 검은색(`text-gray-900`)으로 명시적으로 설정하여 사용자가 입력한 내용을 더 명확하게 볼 수 있도록 개선했습니다.

## 🎨 변경 사항

### 색상 변경 규칙

- **입력 텍스트**: `text-gray-900` 추가 (검은색)
- **Placeholder 텍스트**: `placeholder:text-gray-500` 추가 (회색)

### 변경된 파일 목록

#### 1. Step1BasicInfo.tsx
- 플랜 이름 입력 필드
- 학습 기간 시작일/종료일 입력 필드
- 블록 세트 이름 입력 필드 (생성/수정)
- 추가 기간 시작일/종료일 입력 필드

#### 2. 패널 컴포넌트들

**ExclusionsPanel.tsx**
- 제외일 사유 입력 필드

**AcademySchedulePanel.tsx**
- 학원 일정 시작/종료 시간 입력 필드
- 학원 이름 입력 필드
- 과목 입력 필드
- 이동시간 입력 필드 (number)

**NonStudyTimeBlocksPanel.tsx**
- 시작/종료 시간 입력 필드

#### 3. 공유 컴포넌트들

**ContentSelector.tsx**
- 콘텐츠 검색 입력 필드

**ContentMasterSearch.tsx**
- 제목 검색 입력 필드
- 과목 검색 입력 필드

**RecommendedContentsPanel.tsx**
- 추천 개수 입력 필드 (number)

## 📝 변경 예시

### 변경 전
```tsx
<input
  type="text"
  className="w-full rounded-lg border border-gray-300 px-3 py-2 text-sm focus:border-gray-900 focus:outline-none"
  placeholder="예: 1학기 중간고사 대비"
/>
```

### 변경 후
```tsx
<input
  type="text"
  className="w-full rounded-lg border border-gray-300 px-3 py-2 text-sm text-gray-900 placeholder:text-gray-500 focus:border-gray-900 focus:outline-none"
  placeholder="예: 1학기 중간고사 대비"
/>
```

## 🎯 개선 효과

### 가독성 향상

1. **텍스트 명확성**: 입력한 텍스트가 검은색으로 표시되어 배경과의 대비가 증가하여 읽기 쉬워졌습니다.

2. **일관성 개선**: 모든 입력 필드에 동일한 색상 규칙을 적용하여 일관된 사용자 경험을 제공합니다.

3. **접근성 향상**: 시력이 약한 사용자도 입력한 내용을 더 쉽게 확인할 수 있습니다.

### 수정된 컴포넌트

- ✅ Step1BasicInfo (기본 정보 입력)

- ✅ NonStudyTimeBlocksPanel (학습 시간 제외 항목)

### 입력 필드 유형

- `type="text"`: 텍스트 입력 필드
- `type="date"`: 날짜 선택 필드
- `type="time"`: 시간 선택 필드
- `type="number"`: 숫자 입력 필드

## 🔍 색상 가이드라인

### 입력 필드 텍스트 색상

1. **입력 텍스트**: `text-gray-900` (매우 진한 회색/검은색)
2. **Placeholder**: `placeholder:text-gray-500` (중간 회색)

### 적용 원칙

- 모든 입력 필드에 `text-gray-900` 명시적 추가
- Placeholder가 있는 필드에 `placeholder:text-gray-500` 추가
- disabled 상태는 기존 스타일 유지 (opacity 적용)
