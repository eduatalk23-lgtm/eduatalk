## Q21. 오류→재탐구 좋은 사례

**리서치에서 발견된 실제 합격 사례:**

> 융합과학 탐구에서 빌딩풍을 연구한 학생이 가설과 다른 결과에 대해 "기온을 지목했고, 외부와 실험공간의 단절 실패로 공기 팽창 시의 영향을 정확히 측정할 수 없었다"고 **오차 원인을 분석**하며 한계를 인식한 것이 높이 평가됨.

**추가 예시:**
> 과학 세특에서 "기체 확산 실험에서 예외 결과가 나타난 원인을 분자 운동 이론으로 재해석함"

## Q22. 진로교과 vs 비진로교과 세특 차이

| 기준 | 진로교과 세특 | 비진로교과 세특 |
|---|---|---|
| 탐구 주제의 진로 연결 | **가산** (필수는 아님) | 필요 없음. 해당 교과 역량 중심 |
| 탐구 깊이 기대 수준 | **높음** — 전공 연계 심화 탐구 기대 | 보통 — 교과 성취기준 충족이면 충분 |
| 결론에서 진로 제언 | 가산 (자연스러울 때만) | 불필요 |
| 8단계 흐름 충족 수준 | **엄격** — 탐구 과정이 명확해야 | 느슨 — 학업태도/참여도 중심도 가능 |

**입학사정관 원문:**
- 한양대: "전공적합성이 아닌 **계열적합성**에 초점. 물리 역량을 보여준 학생이 화학과를 지원해도 과학 교과 역량으로 동일하게 평가"
- 가천대: "진로와 관계가 있든 없든 평가에 반영. 비진로 교과에서는 학생 관심사가 보이면 충분"

## Q23. 비진로교과 세특의 좋은 방향

| 교과 | 좋은 방향 |
|---|---|
| 국어/영어 | 논리적 표현력, 수사적 기법 분석, 비판적 독해. 관심 분야는 "소재"로만 활용 |
| 체육 | 리더십, 팀워크, 자기관리 역량 → 인성과 공동체 역량 평가에 기여 |
| 음악/미술 | 창의성, 감수성, 표현력 → 다양한 역량을 가진 학생이라는 인상 |
| 수학 | 교과 핵심역량(추론, 문제해결, 모델링) 중심 |

**실전 비율** (전문가 권장):
- 진로 적극 연결: 2~3과목
- 자연스럽게 관심사 녹이기: 2~3과목
- 교과 역량만 100%: 4~5과목 (진로 연결 불필요)

## Q25. 동아리활동 세특 좋은 방향

기존 정의에 추가/수정:
```
진로 관심에서 출발 → 동아리 내 역할+기여 → 전공 심화 탐구 수행
→ 결과물/산출물 → 동료와의 협업/소통 → 후속 활동으로 연결
```
**추가 요소:**
1. **2년 이상 지속적 활동** (매년 바뀌면 감점)
2. **비판적사고역량, 창의적사고역량, 자기주도역량, 소통/협업역량**을 횡단 평가
3. 동아리 내에서 **문제를 설정하고 해결하는 과정**이 핵심
4. 단순 참여("즐겁게 참여함")가 아닌 **역할, 기여, 확장** 명시

## Q26. 진로활동 세특 좋은 방향

**베스트 구성** (진학 전문가 분석):
```
학교 활동 참여 → 호기심 발생 → 직접 조사/실험 설계(자기주도적 양적 조사 ~300자)
→ 심화 탐구 활동 (~250자) → 진로 계획 구체화 (~150자)
```
**주의:** 진로검사 결과, 학과탐방, 박람회 참여 등 단기 참여활동은 "참고 자료 정도"로만 활용

## Q27. 동아리/진로에 8단계 적용 여부

**답변: (b) 일부 적용 — 핵심 흐름은 동일하되 세특만큼 엄격하지 않음**

- 동아리/진로: **"궁금증 → 탐구 → 깨달음"의 사고 흐름**이 자연스럽게 드러나면 충분
- 세특 대비 차이:
  - 동아리는 **"협업 과정"**이 추가 평가
  - 진로활동은 **"진로와의 연결성"**이 추가 평가
  - ⑧ 오류→재탐구는 있으면 가산이지만 필수는 아님

## Q33. 기타 — 리서치에서 발견된 추가 사항

1. **교차 비교 평가**: 입학사정관은 같은 학교 동일 과목 수강한 학생들의 세특을 교차 비교. 여러 교사가 같은 학생의 동일 능력을 공통 증언하면 신뢰도 크게 상승

2. **2028대입 변화**: 5등급제 전환 + 성취평가제 확대 → 내신 변별력 약화 → **정성평가+대학별 고사 비중 더욱 확대 전망**

3. **대교협 유사도 검증시스템**: 전국 대학 지원자 자료의 유사도를 자동 검증. 타 학교/이전년도와 유사도 높으면 소명 요구 → 교사 면담

4. **학교폭력 조치사항**: 2026학년도부터 전 대학 전 전형 반영. 2호 2% ~ 9호 50% 정량 감점
