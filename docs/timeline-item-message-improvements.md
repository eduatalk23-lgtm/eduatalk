# 타임라인 아이템 메시지 개선

## 변경 사항

점심시간과 자율학습 영역의 시간 표시를 친근한 문구로 변경하고, 자율학습 배지 색상을 배경과 일치하도록 수정했습니다.

### 개선 내용

1. **점심시간 영역**
   - 기존: 시간 표시 (예: "12:00 ~ 13:00")
   - 변경: "맛있는 점심식사 드세요." 문구 표시
   - 헤더의 시간 표시 제거

2. **자율학습 영역**
   - 기존: 시간 표시 (예: "19:00 ~ 22:00")
   - 변경: "완료하지 못한 학습 또는 복습을 진행하세요." 문구 표시
   - 헤더의 시간 표시 제거

3. **자율학습 배지 색상**
   - 기존: `bg-orange-500` (주황색)
   - 변경: `bg-green-500` (초록색) - 배경색(`bg-green-50`)과 일치

### 변경 파일

- `app/(student)/plan/calendar/_components/TimelineItem.tsx`
  - 점심시간일 때 특별한 메시지 표시
  - 자율학습일 때 특별한 메시지 표시
  - 자율학습 배지 색상을 `bg-green-500`로 변경
  - 점심시간과 자율학습일 때 헤더의 시간 표시 제거
