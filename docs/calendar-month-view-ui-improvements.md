# 캘린더 월별 뷰 UI 개선

## 작업 일시
2024년 12월 15일

## 개선 사항

### 1. CalendarPlanCard Compact 모드 개선 ✅

#### 텍스트 오버플로우 문제 해결
- **문제**: 플랜 텍스트가 셀 밖으로 나가는 문제
- **해결**:
  - 교과/과목과 회차 정보를 한 줄에 통합 표시
  - `truncate` 클래스로 텍스트 잘림 처리
  - `title` 속성으로 전체 텍스트 툴팁 제공
  - `min-w-0` 클래스로 flex 아이템 오버플로우 방지

#### 레이아웃 개선
- **변경 전**: 가로 배치 (아이콘 + 교과 + 회차 + 상태)
- **변경 후**: 세로 스택 구조로 단순화
  - 1행: 아이콘 + 교과/과목(회차 포함) + 상태 뱃지
  - 더 효율적인 공간 활용

#### 시각적 개선
- 상태 뱃지 간소화: ✅ → ✓, ⏱️ → ⏱ (이모지 → 심볼)
- 패딩 조정: `p-1 py-0.5` → `p-1.5 py-1` (약간 증가)
- 텍스트 크기: `text-[10px]` → `text-[11px]` (가독성 향상)
- 툴팁 개선: 교과, 회차, 제목 정보 모두 표시

### 2. 월별 뷰 날짜 셀 레이아웃 개선 ✅

#### 간격 및 크기 조정
- 날짜 헤더 간격: `gap-1.5` → `gap-1` (더 컴팩트)
- 날짜 숫자 크기: `text-lg` → `text-base md:text-lg` (반응형)
- 오늘 표시 점 크기: `text-xs` → `text-[10px]` (더 작게)
- 날짜 타입 배지 크기: `text-sm` → `text-[10px]` (일관성)

#### 타임슬롯 표시 개선
- 학원일정/특수 슬롯 스타일 개선:
  - 다크모드 지원 추가
  - border 추가로 시각적 구분 강화
  - 텍스트 크기: `text-xs` → `text-[10px]` (일관성)
- "더 있음" 표시 개선:
  - `⋯` → `+N` 형식으로 변경 (명확한 정보 제공)
  - 다크모드 지원
  - border 추가

#### 구조 개선
- `min-w-0` 클래스 추가로 오버플로우 방지
- flex 컨테이너에 `min-w-0` 적용
- 중첩된 flex 구조 최적화

### 3. 시각적 계층 구조 개선 ✅

#### 일관된 텍스트 크기
- 모든 컴팩트 요소: `text-[10px]` 또는 `text-[11px]` 사용
- 날짜 헤더: `text-base md:text-lg` (반응형)
- 일관된 크기 체계로 가독성 향상

#### 색상 및 테두리
- 모든 타임슬롯에 border 추가
- 다크모드 색상 지원 강화
- 시각적 계층 구조 명확화

#### 간격 최적화
- 요소 간 간격: `gap-1` (더 컴팩트)
- 카드 내부 패딩: `p-1.5 py-1` (적절한 여백)

## 기술적 세부사항

### 텍스트 오버플로우 처리
```tsx
// min-w-0으로 flex 아이템 오버플로우 방지
<div className="flex items-center gap-1 min-w-0">
  <span className="truncate font-medium min-w-0 flex-1">
    {fullText}
  </span>
</div>
```

### 툴팁 구현
```tsx
const tooltipText = [
  plan.contentSubjectCategory || plan.contentSubject || "과목 없음",
  plan.contentEpisode && `회차: ${plan.contentEpisode}`,
  plan.contentTitle && `제목: ${plan.contentTitle}`,
]
  .filter(Boolean)
  .join(" | ");

<div title={tooltipText}>
```

### 반응형 텍스트 크기
```tsx
// 모바일: text-base, 데스크톱: text-lg
<div className="text-base md:text-lg font-bold">
```
