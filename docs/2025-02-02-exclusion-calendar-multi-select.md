# ExclusionManagement 달력 형태 날짜 선택 개선

## 작업 일시
2025-02-02

## 작업 내용

### 요구사항
- 비연속 다중 선택 모드에서 리스트 형태의 날짜 선택을 달력 형태로 변경
- 선택된 날짜가 목록에 쌓여서 표시되도록 개선

### 구현 내용

#### 1. MultiSelectCalendar 컴포넌트 생성
- **위치**: `app/(student)/blocks/_components/MultiSelectCalendar.tsx`
- **기능**:
  - 달력 형태의 날짜 선택 UI
  - 비연속 다중 선택 지원 (날짜 클릭 시 토글)
  - 이미 제외된 날짜는 선택 불가로 표시
  - 월 네비게이션 (이전/다음 달 이동)
  - 다크모드 지원
  - 오늘 날짜 하이라이트

#### 2. ExclusionManagement 컴포넌트 수정
- **변경 사항**:
  - `exclusionInputType === "multiple"`일 때 리스트 대신 달력 컴포넌트 사용
  - 선택된 날짜를 태그 형태로 목록에 표시
  - 각 태그에 삭제 버튼 추가 (X 아이콘)
  - 전체 삭제 버튼 추가

#### 3. 주요 기능
- **달력 선택**: 날짜를 클릭하면 선택/해제 토글
- **선택된 날짜 목록**: 달력 아래에 선택된 날짜를 태그로 표시
- **개별 삭제**: 각 태그의 X 버튼으로 개별 삭제 가능
- **전체 삭제**: "전체 삭제" 버튼으로 한 번에 모두 제거
- **제외된 날짜 표시**: 이미 제외된 날짜는 회색으로 표시되고 선택 불가

### 기술 스택
- `date-fns`: 날짜 처리 및 달력 렌더링
- `lucide-react`: 아이콘 (ChevronLeft, ChevronRight, X)

### 파일 변경 내역
- **신규 파일**:
  - `app/(student)/blocks/_components/MultiSelectCalendar.tsx`
- **수정 파일**:
  - `app/(student)/blocks/_components/ExclusionManagement.tsx`

### UI 개선 사항
1. **달력 UI**: 직관적인 달력 형태로 날짜 선택
2. **선택된 날짜 목록**: 태그 형태로 선택된 날짜를 한눈에 확인
3. **반응형 디자인**: 모바일/데스크톱 모두 지원
4. **다크모드**: 다크모드 완전 지원

### 사용자 경험 개선
- 기존: 긴 리스트에서 스크롤하며 날짜 찾기
- 개선: 달력에서 한눈에 날짜 확인 및 선택
- 선택된 날짜를 태그로 표시하여 선택 상태를 명확히 확인 가능
