# 역량 루브릭 42개 세부질문 — 컨설턴트 검토 요청서

> **목적**: AI가 역량 등급을 산출할 때 사용하는 42개 세부질문을 검토하여 실제 입시 평가와의 정합성을 확인합니다.
> **읽는 법**: 각 역량 항목별로 현재 질문과 평가 대상을 정리했습니다. 질문 수정/추가/삭제 의견을 주세요.
> 
> **2차 피드백에서 이미 수정된 평가 대상(evalTarget)**: 반영 완료 표시 (✅)

---

## 사용 방식

AI는 세특/창체/행특 텍스트를 분석할 때, 아래 질문 각각에 대해 **A+/A-/B+/B/B-/C** 등급을 매기고, 항목 종합 등급을 산출합니다. 따라서 질문이 정확해야 AI의 등급 판단이 정확합니다.

---

## 1. 학업 영역 (3항목, 10개 질문)

### 1-1. 학업성취도 (academic_achievement)

**평가 대상** ✅: 전체교과평균내신, 주요교과평균내신, 성적 추이, 전공 관련 교과 성취

| # | 현재 질문 | 비고 |
|---|---|---|
| Q0 | 대학 수학에 필요한 기본 교과목의 교과성적은 적절한가? | |
| Q1 | 기본 교과목 이외 과목 성적은 어느 정도인가? | |
| Q2 | 유난히 소홀한 과목이 있는가? | |
| Q3 | 학기별/학년별 성적의 추이는 어떠한가? | |

> ❓ 질문 수정/추가/삭제 필요한가요?
> 예: "전체교과평균내신과 주요교과평균내신의 차이가 유의미한가?" 같은 질문을 추가해야 하나요?

### 1-2. 학업태도 (academic_attitude)

**평가 대상**: 수업 참여, 과제 성실성, 질문 태도

| # | 현재 질문 | 비고 |
|---|---|---|
| Q0 | 성취동기와 목표의식을 가지고 자발적으로 학습하려는 의지가 있는가? | |
| Q1 | 새로운 지식 획득을 위해 자기주도적으로 노력하고 있는가? | |
| Q2 | 교과 수업에 적극 참여하여 이해하려는 태도와 열정이 있는가? | |

> ❓ 모델 비교에서 Q2(수업 참여)가 핵심 쟁점이었습니다.
> "발표·질문·토론" 구체적 근거가 없으면 B로 평가하는 것이 맞나요?
> Q2를 "수업 중 발표·질문·토론 등 적극적 참여 행동이 구체적으로 관찰되는가?"로 수정해야 하나요?

### 1-3. 탐구력 (academic_inquiry)

**평가 대상** ✅: 진로연계보고서 및 발표, 실험을 통한 이론학습, 오류분석 후 재검증

| # | 현재 질문 | 비고 |
|---|---|---|
| Q0 | 교과와 각종 탐구활동 등을 통해 지식을 확장하려고 노력하고 있는가? | |
| Q1 | 교과와 탐구활동에서 구체적인 성과를 보이고 있는가? | |
| Q2 | 교내 활동에서 학문에 대한 열의와 지적 관심이 드러나고 있는가? | |

> ❓ 2차 피드백에서 evalTarget에 "실험→이론학습, 오류분석→재검증"이 추가되었는데,
> 질문에도 이를 반영해야 하나요?
> 예: "탐구 과정에서 오류를 분석하고 재검증하여 결론을 도출한 경험이 있는가?" 추가?

## 2. 진로 영역 (3항목, 8개 질문)

### 2-1. 전공 관련 교과 이수 노력 (career_course_effort)

**평가 대상**: 과목 선택의 적합성

| # | 현재 질문 | 비고 |
|---|---|---|
| Q0 | 전공 관련 과목을 적절하게 선택하고 이수한 과목은 얼마나 되는가? | |
| Q1 | 이수하기 위하여 추가적인 노력을 하였는가? | |
| Q2 | 선택과목은 교과목 학습단계에 따라 이수하였는가? | |

> ❓ 모델 비교에서 "기하+데이터과학+AI까지 선택한 점이 매우 적절"이 A- 근거였습니다.
> "전공 관련 심화 과목(기하, 데이터과학, AI 등)을 적극적으로 이수했는가?" 같은 질문을 추가해야 하나요?

### 2-2. 전공 관련 교과 성취도 (career_course_achievement)

**평가 대상** ✅: 전공 관련 과목 성적, 전공과목 성적 추이

| # | 현재 질문 | 비고 |
|---|---|---|
| Q0 | 전공 관련 과목의 성취수준은 적절한가? | |
| Q1 | 동일 교과 내 일반선택 대비 진로선택 성취수준은? | |

> ❓ "전공과목 성적 추이"가 evalTarget에 추가되었는데, 질문도 추가해야 하나요?
> 예: "전공 관련 과목 성적이 학기별로 향상/유지되고 있는가?"

### 2-3. 진로 탐색 활동과 경험 (career_exploration)

**평가 대상** ✅: 진로탐색을 위한 창의적체험활동과 교과활동

| # | 현재 질문 | 비고 |
|---|---|---|
| Q0 | 자신의 관심 분야나 흥미와 관련한 다양한 활동에 참여하여 노력한 경험이 있는가? | |
| Q1 | 교과 활동이나 창체에서 전공에 대한 관심을 가지고 탐색한 경험이 있는가? | |

> ❓ 모델 비교에서 "명시적 진로 목표 서술 부재"가 B+ 근거였습니다.
> "'왜 이 전공인가'에 대한 서사가 생기부에 드러나는가?" 같은 질문을 추가해야 하나요?

## 3. 공동체 영역 (4항목, 10개 질문)

### 3-1. 협업과 소통능력 (community_collaboration)

**평가 대상** ✅: 팀원과 역할분담 및 토론, 지식을 시각화·데이터화하여 이해하기 쉽게 발표하는 능력

| # | 현재 질문 | 비고 |
|---|---|---|
| Q0 | 단체 활동에서 서로 돕고 함께 행동하는 모습이 보이는가? | |
| Q1 | 공동의 과제를 수행하고 완성한 경험이 있는가? | |
| Q2 | 타인의 의견에 공감·수용하며 자신의 정보와 생각을 잘 전달하는가? | |

> ❓ evalTarget에 "지식을 시각화·데이터화하여 발표"가 추가되었습니다.
> Q2를 수정하거나, 별도 질문을 추가해야 하나요?

### 3-2. 나눔과 배려 (community_caring)

**평가 대상** ✅: 멘토멘티활동, 봉사, 동료 지원, 과목부장·학습부장으로서 학습자료 지원

| # | 현재 질문 | 비고 |
|---|---|---|
| Q0 | 학교생활 속에서 나눔을 실천하고 생활화한 경험이 있는가? | |
| Q1 | 타인을 위하여 양보하거나 배려를 실천한 구체적 경험이 있는가? | |
| Q2 | 상대를 이해하고 존중하는 노력을 기울이고 있는가? | |

> ❓ "멘토멘티활동, 과목부장·학습부장 학습자료 지원"이 추가되었습니다.
> "멘토링이나 학습부장 활동 등을 통해 동료의 학습을 지원한 경험이 있는가?" 추가?

### 3-3. 성실성과 규칙준수 (community_integrity)

**평가 대상**: 출결, 과제, 기타과목 관리

| # | 현재 질문 | 비고 |
|---|---|---|
| Q0 | 교내 활동에서 자신이 맡은 역할에 최선을 다하려고 노력한 경험이 있는가? | |
| Q1 | 자신이 속한 공동체가 정한 규칙과 규정을 준수하고 있는가? | |

> ❓ 수정 필요한가요?

### 3-4. 리더십 (community_leadership)

**평가 대상** ✅: 임원, 교과목 부장, 자치활동, 임원이 아니더라도 스스로 주도하여 친구들을 이끈 경험

| # | 현재 질문 | 비고 |
|---|---|---|
| Q0 | 공동체의 목표를 달성하기 위해 계획하고 실행을 주도한 경험이 있는가? | |
| Q1 | 구성원들의 인정과 신뢰를 바탕으로 참여를 이끌어내고 조율한 경험이 있는가? | |

> ❓ "임원이 아니더라도 주도한 경험"이 evalTarget에 추가되었습니다.
> "공식적 직책 없이도 스스로 나서서 친구들을 이끌어 프로젝트를 진행한 경험이 있는가?" 추가?

## 요약

**현재**: 10개 역량 × 2~4개 질문 = **32개 질문** (42개가 아닌 32개로 확인됨)
**검토 요청**: 각 항목의 질문이 실제 입시 평가와 맞는지 + 추가/삭제/수정 의견

**의견 형식**: "1-3 Q2: '~로 수정해주세요'" 또는 "2-3: Q2 추가해주세요: '~'" 형태로 주시면 바로 반영합니다.
